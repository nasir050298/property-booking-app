# Property Listings React Application

A modern property listing application built with React that allows users to browse and filter properties.

## Features

- **Property Listings**: Display properties with images, descriptions, and details
- **Advanced Filtering**: Filter by location, superhost status, and property type
- **Responsive Design**: Works on desktop and mobile devices
- **Reusable Components**: Modular component architecture

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies:
   ```bash
   npm install
   ```
